# Portfolio Summary

## Project Name

UA Playable Games Lab

## Short Description

A beginner-friendly HTML5 playable ads learning project built with Vite, TypeScript, and Phaser 3.

## Project Purpose

This project was created to learn the fundamentals of UA playable ads by building multiple small interactive game prototypes.

## Main Features

- Tap Monster playable prototype
- Runner Gate playable prototype
- Merge Cannon playable prototype
- Gem Collector playable prototype
- Shared end card flow
- CTA and replay buttons
- HTML5 ZIP export for Tap Monster
- MRAID and ad network learning notes
- Git version workflow with release tags

## Technologies Used

- Vite
- TypeScript
- Phaser 3
- HTML
- CSS
- JSZip
- Git
- GitHub

## Playable Ads Concepts Practiced

- Tap interaction
- Drag interaction
- Drag-and-drop merge
- Collision detection
- Enemy HP
- Projectile targeting
- Reward system
- Countdown timer
- End card
- CTA button
- Replay flow
- HTML5 playable export
- MRAID notes

## CV Bullet Points

- Built multiple HTML5 playable ad prototypes using Phaser 3 and TypeScript.
- Implemented tap, drag, drag-and-drop, merge, collision, projectile, reward, timer, CTA, and end-card mechanics.
- Created a reusable scene architecture with shared scene keys, UI helpers, and game registry.
- Added a standalone HTML5 ZIP export workflow for a playable ad prototype.
- Managed the project with Git feature branches, semantic commits, changelog entries, and version tags.
